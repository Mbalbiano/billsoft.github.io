# Billsoft

Proyecto grupal desarrollado para la materia Diseño y Desarrollo Web.

## Descripción
Billsoft es una propuesta de sitio web orientada a la gestión financiera personal. La idea principal es ofrecer una experiencia simple, clara y visual para que los usuarios puedan registrar sus gastos, organizar sus movimientos y llevar un mejor control de su economía.

## Objetivo del proyecto
Crear una interfaz web funcional y atractiva que permita:
- presentar la propuesta de la aplicación;
- mostrar los beneficios de una herramienta de control financiero;
- permitir el registro e inicio de sesión de usuarios;
- visualizar gastos e ingresos de manera organizada;
- mostrar un resumen mensual de la actividad financiera.

## Funcionalidades incluidas
- Página de inicio con diseño moderno y responsive.
- Sección “Sobre nosotros” con misión, visión y valores.
- Formularios de registro y login.
- Vista de ingresos y gastos con tabla de movimientos.
- Vista de resumen mensual.
- Validaciones básicas de formularios con JavaScript.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript vanilla

## Estructura del proyecto
- index.html: página principal del sitio.
- seccion/: archivos de las diferentes secciones, como registro, login, gastos, resumen mensual y sobre nosotros.
- assets/css/: estilos personalizados del proyecto.
- assets/JS/: lógica de validación y comportamiento básico del frontend.

## Cómo visualizar el proyecto
1. Clonar o descargar este repositorio.
2. Abrir el archivo index.html en tu navegador.
3. También puedes publicar el proyecto en GitHub Pages para verlo en línea.

## Estado del proyecto
El proyecto se encuentra en etapa de desarrollo inicial, con una interfaz funcional y un enfoque visual orientado a demostrar la idea de Billsoft como herramienta de control financiero.
